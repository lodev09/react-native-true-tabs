# react-native-true-tabs

Flexible native tabs for React Native

## Installation


```sh
npm install react-native-true-tabs
```


## Usage


```js
import { TrueTabsView } from "react-native-true-tabs";

// ...

<TrueTabsView color="tomato" />
```
